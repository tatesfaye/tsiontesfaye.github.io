# Model Hotel Booking/Cancelation

How do hotels maximize booking while planning for potential cancellations? This repo answers that question by modeling the probability of guests canceling a booking in the [hotel_bookings.md](Modeling/hotel_bookings.md) workbook. The resulting model has an 81% predictive accuracy.
  
  - **_Skills_**: Data Analysis, Modeling
  - **_Tools/Packages_**: tidymodels, tidyverse
